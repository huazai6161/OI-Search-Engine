水题解

首先不难想到一个忐忑心：从最左（初始点）跳到最右，再跳到第二左，再跳到第二右……以此类推。但是这仅仅是一个感性理解上路径最远的走法，它需要一个证明。

以下不妨设 $0 = h_0 < h_1 < h_2 < h_3 < \cdots < h_n$。注意到，我们其实仅需要证明最优情况下第 $1$ 步跳到 $h_n$ 即可（此后左右翻转可以证明第二步跳到 $h_1$，以此类推，整个过程的正确定性证明也就完成了）。

假设存在一种最优情况使得第一步不是从 $0$ 跳到 $h_n$ 而是跳到 $h_p \ (1 \leq p \leq n - 1)$，那么我们可以分两种情况讨论：

1. $h_n$ 是终点

那么我们第一步改变 $h_n$ 然后全程反转，则体力值耗费增加 $h_n^2 - h_p^2 > 0$，矛盾。

2. $h_n$ 不是终点。

不妨 $h_n$ 跳向了 $h_q$。

那么我们第一步去跳 $h_n$，然后从 $h_p$ 到 $h_n$ 的跳跃全程反转，然后 $h_p$ 跳 $h_q$，则体力耗费增加 $h_p^2 + (h_p - h_q)^2 - h_p^2 - (h_n - h_q)^2 = 2h_q(h_n - h_p) > 0$，矛盾。

综上，第一步一定是从 $0$ 跳到 $h_n$，证毕。
