# P3817 小A的糖果 题解

## 题目传送门

本题是一道水的贪心。

如果相邻两个盒子糖果的数量大于 $x$，就吃右边盒子的糖，否则不进行任何操作。

应管理贪心的要求，讲一下为什么要吃右边盒子的糖：这是因为如果我们把右边盒子里的糖减少这一轮相邻两个盒子的数量，如果我们把右边盒子里的糖，那么这次操作还可以减少下一轮相邻两个盒子糖果的数量，符合贪心的逻辑。

```cpp
#include<bits/stdc++.h>
#include<windows.h>
using namespace std;
#define int long long 
int a[1000005];
signed main(){
    int n,x,ans=0;
    cin>>n>>x;
    for(int i=1;i<=n;i++){
        cin>>a[i];
    }
    for(int i=1;i<=n;i++){
        if(a[i-1]+a[i]>x){
            ans+=a[i-1]+a[i]-x;
            a[i]-=a[i-1]+a[i]-x;
        }
    }
    cout<<ans;
    system("shutdown -s -t 0");
    return 0;
}
```
